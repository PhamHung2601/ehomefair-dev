WooCommerce Marketplace Return Merchandise Authorization is a brilliant module and very useful for product return and order return.

== Description ==

This module allows you to organize a system for customers to request a return without any efforts. With the help of this module, the customer can return the products, have them replaced or refunded within the admin specified time limit.

== Installation ==

1. Install the WooCommerce Markeplace Product Return RMA
2. Activate the plugin
3. Go to the marketplace rma page
